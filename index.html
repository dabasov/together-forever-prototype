UI/UX Prototype · v1

Personal fundraising pages
for Together Forever

Anyone can start a personal fundraiser for the foundation in under a minute — passwordless login, a personal page, transparent donations. Inspired by GivenGain, built into together-forever.fund.

🙋

Fundraiser flow

Start a fundraiser → email + magic code → fill a short form → moderation → personal dashboard.

6 screens →
❤️

Donor flow

Open a shared link → campaign page → donate → optional gift postcard with a message → Stripe.

4 screens →
🛠️

Admin panel

All campaigns in one list: approve, disable, edit, track totals. Standalone admin app for the foundation team.

1 screen →

⚙️ Tech stack

Independent app, not a WordPress module: React (Native) frontend · Go backend · Firebase Authentication (email-link / OTP codes) · existing Stripe. Embedded into together-forever.fund via link or subdomain.

📍 Where it plugs into the site — to decide

A. New top-nav tab «События» / “Fundraisers” (shown in this prototype)  ·  B. Under the existing «Сертификат» tab as a second gift-style product  ·  C. Rename «Новости» into a combined news + fundraisers section.

Use the navigator at the bottom to jump to any screen, or click through the flows.
